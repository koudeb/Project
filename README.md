## ⚠️ Issues Faced & Solutions

### Issue 1: Cursor login not working
- Problem: Clicking login did not open browser
- Solution: Tried setting default browser and retrying multiple times

### Issue 2: Git Clone option not showing
- Problem: "Git: Clone" was missing
- Solution: Installed Git and restarted Cursor

### Issue 3: Windows Defender blocked Cursor
- Problem: Security warning appeared
- Solution: Verified source and allowed access
